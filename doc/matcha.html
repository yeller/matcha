run-match
(run-match matcher a)
runs a matcher, given a value to match against.
Returns a map:

if the matcher matches the value:
{:pass? true}

if the matcher fails:
{:pass? false
 :matcher matcher
 :expected (a string)
 :was (a string)}

the results can be made human readable with (format-message result)
